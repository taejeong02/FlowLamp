# Flow Lamp

## 1. 프로젝트 소개
Flow Lamp는 학습 환경에서 조명의 전원, 밝기, 위치를 더 편리하게 제어하기 위해 개발한 IoT 스마트 조명 프로젝트입니다. Flutter 앱과 FastAPI 서버, Raspberry Pi 5, 카메라, LED, Dynamixel 모터를 연동하여 사용자의 조작과 감지 결과에 따라 조명의 상태를 제어하도록 구현했습니다.

사용자는 앱을 통해 전원, 밝기, 위치, 야간 모드를 설정할 수 있으며, Raspberry Pi는 전달받은 명령을 기반으로 하드웨어를 제어합니다. 또한 카메라 기반 사용자 감지를 통해 사람이 감지되면 대기 모드에서 기본 모드로 전환되도록 설계했습니다.

이 프로젝트는 앱 개발, API 서버 구축, 임베디드 제어, 하드웨어 연동을 함께 경험한 IoT 기반 캡스톤 프로젝트입니다.

## 2. 개발 배경
책상에서 공부하거나 작업할 때 조명의 위치와 밝기는 사용자의 집중도와 시야 확보에 큰 영향을 줍니다. 하지만 일반적인 책상 조명은 사용자가 직접 전원을 켜고, 밝기와 각도를 조절해야 하며, 상황에 따라 반복적으로 조작해야 하는 불편함이 있습니다.

특히 조명 위치가 적절하지 않으면 손이나 물체로 인해 그림자가 생기거나, 빛이 눈에 직접 들어와 피로감을 유발할 수 있습니다. 또한 사용자가 자리를 비우거나 다시 착석했을 때 조명을 매번 수동으로 조작해야 하기 때문에 학습 흐름이 끊길 수 있습니다.

Flow Lamp는 이러한 문제를 해결하기 위해 사용자의 존재 여부를 감지하고, 앱을 통해 조명의 밝기와 위치를 쉽게 제어할 수 있는 스마트 조명 시스템으로 기획되었습니다. 이를 통해 사용자가 조명을 직접 조작하는 과정을 줄이고, 더 편리하고 집중하기 좋은 학습 환경을 제공하는 것을 목표로 했습니다.

## 3. 주요 기능
Flow Lamp는 앱 제어, 사용자 감지, 조명 제어, 모드 전환 기능을 중심으로 구성되어 있습니다. 사용자는 Flutter 앱을 통해 조명의 상태를 직접 제어할 수 있으며, Raspberry Pi는 앱에서 전달된 명령과 카메라 감지 결과를 바탕으로 LED와 모터를 제어합니다.

### 3.1 앱 기반 조명 제어
Flutter 앱을 통해 조명의 전원, 밝기, 위치, 야간 모드를 제어할 수 있습니다.

- 조명 전원 ON/OFF
- 조명 밝기 조절
- 조명 색상 조절(RGB)
- 조명 위치 조절
- 야간 모드 시간 설정
- 알람 설정
- 학습 점수/ 자세 점수 데이터 확인

### 3.2 Raspberry Pi 기반 제어 서버
Raspberry Pi에서 FastAPI 서버를 실행하여 앱과 하드웨어 제어 로직을 연결했습니다. 앱에서 전달된 HTTP 요청을 받아 LED, 모터, 모드 상태를 제어하는 구조로 설계했습니다.

- FastAPI 기반 REST API 서버 구성
- 앱 요청에 따른 조명 상태 변경
- 전원, 밝기, 위치, 모드 상태 관리
- 하드웨어 제어 로직과 API 서버 연동

### 3.3 카메라 기반 사용자 감지
카메라를 활용하여 사용자의 존재 여부를 감지하고, 사용자가 감지되면 대기 모드에서 기본 모드로 전환되도록 설계했습니다.

- 카메라를 통한 사용자 감지
- 사용자 감지 시 Standby Mode에서 Normal Mode로 전환
- 사용자의 착석 여부를 기반으로 학습 점수 관리

### 3.4 LED 밝기 제어
LED 제어 기능을 통해 사용자가 원하는 밝기로 조명을 조절할 수 있도록 구현했습니다. 앱에서 설정한 밝기 값은 Raspberry Pi 서버로 전달되고, 서버는 해당 값을 기반으로 LED 출력을 제어합니다.

- 밝기 값 0~100 범위 제어
- 앱 슬라이더를 통한 밝기 조절
- 밝기 증가 / 감소 명령 처리
- 현재 밝기 상태 관리

### 3.5 Dynamixel 모터 기반 위치 제어
Dynamixel XL430 모터를 활용하여 조명의 위치를 조절할 수 있도록 설계했습니다. 사용자는 앱에서 위치 값을 설정하거나 축 방향으로 이동 명령을 전달할 수 있습니다.

- X, Y, Z 축 기반 위치 제어
- 위치 설정 API 연동
- 축 방향 이동 명령 처리
- 모터 제어 로직과 서버 연동

### 3.6 야간 모드 스케줄 설정
앱에서 야간 모드의 시작 시간과 종료 시간을 설정할 수 있도록 설계했습니다. Raspberry Pi는 설정된 시간과 활성화 여부를 기반으로 야간 모드 동작을 관리합니다.

- 야간 모드 ON/OFF 설정
- 시작 시간 및 종료 시간 설정
- 설정된 시간에 따른 조명 상태 관리
- 야간 모드 상태 조회

### 3.7 모드 관리
Flow Lamp는 대기 모드와 기본 모드를 중심으로 동작합니다. 대기 모드에서는 사용자의 감지를 기다리고, 사용자가 감지되면 기본 모드로 전환되어 조명 제어 기능을 수행합니다.

- Standby Mode: 사용자 감지 대기 상태
- Normal Mode: 조명 제어 동작 상태
- 앱 또는 감지 결과에 따른 모드 전환
- 현재 모드 상태 조회
